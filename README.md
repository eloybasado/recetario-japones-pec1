# PEC1 - Sitio web de gastronomia

Proyecto estatico para la asignatura Herramientas HTML y CSS.

## Requisitos tecnicos usados

- Parcel
- PostHTML con posthtml-include
- PostCSS con postcss-custom-media y autoprefixer
- JavaScript sin frameworks
- CSS sin frameworks

## Scripts

- `npm start`: limpia y lanza entorno de desarrollo con Parcel
- `npm run build`: limpia y genera la build de produccion

## Estructura principal

- `src/index.html`
- `src/categoria.html`
- `src/det1.html`
- `src/det2.html`
- `src/links.html`
- `src/partials/header.html`
- `src/partials/footer.html`
- `src/css/styles.css`
- `src/js/app.js`
- `src/img/`
